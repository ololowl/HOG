Группа 321

Основная часть сделана по описанию. 
Гистограммы считаются в клетках, размеры которых определяются
размерами картинок, и для каждой картинки 7 клеток по горизонтали, 
8 по вертикали.
Нормализуются они в блоках 3х3 келетки. Блоки пересекаются,
для лучшего результата.
Границы изображения отражаются зеркально. Это реализовано в 
методе класса Matrix -- unary_map.

Реализованы обе дополнительные части.
LBP и цветовые признаки.
